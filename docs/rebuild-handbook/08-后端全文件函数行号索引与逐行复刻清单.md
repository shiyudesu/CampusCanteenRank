## 10. 逐行复刻模板（建议你打印后逐项打勾）

你复刻每个函数时按下面模板做：

1. 函数签名先抄出来。
2. 写输入校验。
3. 写主逻辑分支。
4. 写错误映射。
5. 写成功返回。
6. 写一个成功测试。
7. 写一个失败测试。

模板示例：

1. 函数名：CreateComment。
2. 输入：userId、stallId、CreateCommentRequest。
3. 校验：content 非空，rootId/parentId/replyToUserId 合法。
4. 主逻辑：校验 root 和 parent，创建评论。
5. 副作用：回复路径更新根评论 reply_count。
6. 返回：CommentItem。
7. 失败例：rootId 不存在。

---

## 11. 你如何把“逐行学习”落地成代码能力

1. 第一遍：只读并做函数流程图。
2. 第二遍：隐藏源码，按索引从函数签名开始默写。
3. 第三遍：对照测试修正细节。
4. 第四遍：自己增加一个小功能并补测试。

做到第四遍，你已经不是“会看代码”，而是“会写同级别工程代码”。
